# legal-reach

A modern platform connecting clients with qualified legal professionals. Features lawyer discovery, filtering by practice areas, and user dashboards. Built with React, Vite, and TailwindCSS.

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## Project Structure

- **components/** - Reusable React components (About, FeaturedLawyers, Footer, Navigation, PracticeAreas, Slider)
- **pages/** - Page components (Home, Login, Signup, Dashboard, FindLawyers, LawyerDetails, LawyerDashboard)
- **assets/** - Static assets and resources

## Technologies Used

- React 19
- Vite
- React Router DOM
- TailwindCSS
- Lucide React Icons
